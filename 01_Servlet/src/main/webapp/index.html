1. 서블릿

서블릿이란, 웹 서비스를 위한 자바 클래스를 말하며 자바를 사용해서 웹을 만들기 위한 기술이다.
javax.servlet.http.HttpServlet 클래스를 상속받아야 한다.

2. 서블릿 생명주기(Life Cycle)

객체가 생성된 후 소멸되기까지의 과정을 생명주기(Life Cycle)라고 한다
1) 첫 번째 요청일 경우, 객체를 생성하고 init() 메소드를 호출한다. (한 번만 수행된다. 이후 요청에 대해서는 객체를 생성하지 않음)
2) service() 메소드가 사용자의 HTTp 요청에 따라 doGet(), doPost() 메소드를 호출한다.
3) 서블릿이 서비스되지 않거나, 재 컴파일 될 때 서블릿 객체가 destroy() 메소드를 호출 후 소멸한다.

3. HTTP 요청 처리

1) GET 방식 URL에 데이터를 포함시켜 요청하는 것으로 보안 유지가 불가능하다

2) POST 방식은 데이터를 URL에 노출하지 않고 요청하는 것으로 보안 유지가 가능하다.
